
# CRM Авто (PWA) с расчётом зарплаты для Цессии

Добавлен расчёт зарплаты (15% от суммы) только для сделок типа «Цессия».

## Изменения
- В таблице появился столбец «Зарплата».
- Для сделок «Цессия» автоматически считается зарплата = 15% от суммы.
- Для остальных типов сделок поле «Зарплата» пустое.
- Экспорт CSV теперь тоже содержит колонку «Зарплата».
